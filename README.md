# 🩺 Medical Vision Chatbot

A multimodal voice-interactive chatbot designed to assist with medical tasks using audio input, natural language processing, and a friendly web UI.

## 🔍 Features

- 🎙️ Voice input recording with `SpeechRecognition`
- 🧠 AI-powered medical chatbot (e.g., OpenAI / custom LLM integration)
- 🗣️ Text-to-speech output using [ElevenLabs API](https://www.elevenlabs.io/)
- 🌐 Simple and interactive interface built with `Gradio`
